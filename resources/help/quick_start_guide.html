Opening studies

  • File → Open (or toolbar): open files, folders, or volumes. Folders can be searched recursively for DICOM.
  • Use the series navigator at the bottom to switch series; N toggles its visibility.

Layouts and navigation

  • Layouts: keys 1–4, or View → Layout, or right-click the image. 1x1 shows the focused pane only. 1x2 / 2x1 show the row or column containing the focused window. Double-click a pane to expand to 1x1; double-click again to revert.
  • 2x2 Swap: right-click → Swap → swap grid positions (data stays with each view).
  • Slices: ↑ / ↓ (or mouse wheel in slice mode). Series: ← / →.

Window/level and privacy

  • Adjust W/L with the controls under the image, presets from the context menu, or right-drag where enabled.
  • Privacy view masks patient-related tags in the UI only (display). Toggle via View, context menu, or Ctrl+P / Cmd+P.

Tools and more features

  • ROIs / measure / annotations: toolbar modes or keys H R E M S T A (H = crosshair for pixel values and coordinates; R E M S T A = rectangle, ellipse, measure, select, text, arrow). Crosshairs behave as a ROI type.
  • MPR: right-click → Create MPR View… (see user guide link above).
  • Fusion: right panel Image Fusion; full detail in the fusion doc link above.
  • ACR QA: Tools menu → ACR CT / ACR MRI (pylinac). See QA guide link above.
  • Histogram: Ctrl+Shift+H / Cmd+Shift+H.
  • Export: File → Export for images; tag export and customizations under File as well.

Common keyboard shortcuts

  • P / Z — pan / zoom mode   V — reset view   Shift+A — reset all views
  • I — invert image   Space — cycle overlay visibility
  • Ctrl+T — tag viewer   Ctrl+Z / Ctrl+Shift+Z — undo / redo

For the full shortcut list and feature details, use the browser links at the top or the README in the repository.
