# `.cursor` — agents and skills

- **`agents/`** — Custom [subagents](https://cursor.com/docs/context/subagents) (`*.md` with YAML frontmatter).
- **`skills/`** — [Agent Skills](https://cursor.com/docs/context/skills) (`*/SKILL.md`).

## Promote to global

Copy to `~/.cursor/agents/` and `~/.cursor/skills/` when you want the same team available in every project. Prefer keeping **project-specific** path conventions in orchestration prompts or a small project rule.
