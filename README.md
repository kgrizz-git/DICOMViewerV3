# Notes and Ideas

A private repository for storing personal notes, ideas, and useful references.

## Purpose

This repository serves as a centralized location for:
- Personal notes and thoughts
- Project ideas and concepts
- Useful references and resources
- Code snippets and examples
- Documentation and guides

## Structure

For **Cursor** multi-agent workflows, see [AGENTS.md](AGENTS.md) and `.cursor/agents/` / `.cursor/skills/`. Plans, logs, and security assessments use:

- `plans/` — implementation plans (markdown) and **`orchestration-state.md`** (orchestrator-owned run state for multi-agent handoffs)
- `logs/` — documentation review logs and `test-ledger.md`
- `assessments/` — timestamped security assessments

At the repository root, `list-<folder>.md` files summarize files in selected subdirectories (for example `list-ideas.md`, `list-info.md`, `list-.cursor.md`) with links and short descriptions; refresh them using [update-lists-prompt.md](update-lists-prompt.md) when folders change.

Feel free to organize other content as needed. Consider using folders like:
- `projects/` - Project-specific notes
- `resources/` - Useful links and references
- `snippets/` - Code snippets
- `ideas/` - Brainstorming and concept notes
- `templates/` - Reusable templates and boilerplates

## Privacy

This is a private repository intended for personal use only.
