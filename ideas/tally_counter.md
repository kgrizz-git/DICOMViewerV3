# Tally counter app with widget and ability to see avg number each hour

## Concept Overview
- For iOS, maybe Android, maybe eventually Windows/macOS/Linux.
- Widget to increment/decrement
- Thorough analysis and plotting features
    - Seeing average number logged by hour or grouped hours, averaged over a chosen number of days
    - Plot these averages vs longer periods of time (eg, average over a week, plot vs week 1, 2, 3...)
    - Including just seeing total and/or average per day
