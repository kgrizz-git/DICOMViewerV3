# DICOM Viewer V3 Requirements
# 
# This file lists all Python package dependencies required for the DICOM viewer application.
# 
# Installation: pip install -r requirements.txt
#
# Python version (especially Windows):
# - pylinac (ACR QA) requires Python >= 3.10.
# - Prefer 3.11 or 3.12 (same family as CI). Wheels exist for native deps (e.g. pyjpegls).
# - 3.14+ may have no pyjpegls wheel on Windows yet; pip then builds from source and needs
#   Microsoft C++ Build Tools (https://visualstudio.microsoft.com/visual-cpp-build-tools/).
# - Parallels + UNC paths (\\Mac\Home\...) can confuse venv/pip; a local clone under C:\ is
#   often more reliable if you see odd path or pip self-upgrade issues.
#
# Requirements:
# - PySide6: Cross-platform GUI framework for the application interface
# - pydicom: Library for reading, writing, and working with DICOM files
# - numpy: Numerical computing library for image array operations
# - Pillow: Image processing library for image format conversions
# - matplotlib: Plotting library for histogram display
# - openpyxl: Library for reading and writing Excel files (for tag export)
