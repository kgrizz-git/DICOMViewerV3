# Useful Software and Web Tools (Outside Skills and MCPs)

This page tracks helpful standalone tools and websites that are not part of local skills or MCP toolchains.

## AI and App Prototyping

- Google AI Studio: Build and test prompts, structured outputs, and Gemini-powered workflows.
  - https://aistudio.google.com/
- Stitch by Google Labs: Generate UI ideas and app interfaces from prompts and screenshots.
  - https://stitch.withgoogle.com/
- Cline: Open-source AI coding agent extension for VS Code; supports multiple model providers.
  - https://github.com/cline/cline
- OpenRouter: Unified API gateway for routing requests across many LLM providers; includes free-tier models.
  - https://openrouter.ai/

## Self-Hosted and VPS Infrastructure

- VPS options for self-hosted AI tools (OpenClaw, Open WebUI, etc.): low-cost providers good for running open-source LLM front-ends and agents.
  - Hetzner: https://www.hetzner.com/
  - Vultr: https://www.vultr.com/
  - DigitalOcean: https://www.digitalocean.com/

## Testing and QA

- Playwright: End-to-end browser testing framework by Microsoft; supports Chromium, Firefox, and WebKit.
  - https://playwright.dev/

## Design and Visual Collaboration

- Figma: Browser-based UI/UX design and prototyping tool.
  - https://www.figma.com/
- PixelLab: AI-powered pixel art generation and animation tool.
  - https://www.pixellab.ai/
- Tiled: Open-source 2D level editor for tilemaps and game/world layout design.
  - https://www.mapeditor.org/
